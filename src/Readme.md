

Project Name:
DigiTools Platform Assignment


Description:

DigiTools Platform is a simple and user-friendly web application where users can explore different digital tools and use them easily. The goal of this project is to provide a smooth and clean experience so users can browse products, add them to a cart, and manage them without any confusion.

I have used the following Technologies :
React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
React-Toastify
JSON (for product data)


Features:

1. Dynamic Cart System
Users can add and remove products from the cart, and the total updates automatically.

2. Notification System
Toast notifications are used to give feedback when users perform actions like adding or removing items.

3. Responsive Design
The application is fully responsive and works well on mobile, tablet, and desktop devices.

Installation and Setup :

npm install
npm run dev
